#Compteurs
Le projet du chronomètre repose sur 3 compteurs différents:
-Counter_dixieme_RE pour les 1/10s
-Counter_Unit_4b_RE pour les unité des secondes et des minutes
-Counter_Diz_4b_RE pour les dizaines des secondes et des minutes

Ainsi, les deux derniers compteurs sont répliqués et réutilisé dans des parties différentes.
#Entité top
L'entité de plus haut niveau pour les compteurs se trouve dans "/top counter_dixieme_min_sec/"
elle reutilise les trois compteurs définis precedemment
